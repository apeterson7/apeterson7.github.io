Part 1 - Python Intro

Python Evaluator

The Python Evaluator is a a program that runs for the command line, which evaluates bits of python code that you enter. Lets try launching it. Open your command prompt and type python and press enter. Your command prompt should now be >>>. Try entering the following:

  • >>> "hello world"
  • >>> hello world
  • >>> "hello" + " world"
  • >>> "hello" == "world"
  • >>> "hello" == "hello"
  • >>> 2
  • >>> 2 + 3
  • >>> "2 + 3"
  • >>> 2 + 3 == 2
  • >>> 2 + 3 == 5
  • >>> 2 + 3 > 3
  • >>> 2 + 3 > 3
  • Use exit() to quit the Python Evaluator

Critical Thinking Questions: What produced an error? How did the function of + change when you used words and numbers? Can you determine some basic python syntax?

Data Types

Python has a few simple data types that store different types of data, and are manipulated in different ways.

  • Strings: Defined by "". Used to store text. Can be printed, concatonated, compared, and more.
  • Integer: Store numbers, math operators can be applied, can be compared, can be printed
  • Boolean: Can be compared, are produced when two pieces of data of the same type are compared.

Variables

Reopen the python emulator. Now try entering the following

  • >>> message = "Hello "
  • >>> message
  • >>> name = "Barb"
  • >>> message + name
  • >>> message == name
  • >>> note = message + name
  • >>> note
  • >>> apples = 4
  • >>> oranges = 3
  • >>> apples + oranges
  • >>> apples > oranges
  • >>> apples == oranges
  • >>> apples = apples - 1
  • >>> apples == oranges
  • >>> worldIsFlat = False
  • >>> obamaIsPres = True
  • >>> worldIsFlat or obamaIsPres
  • >>> worldIsFlat and obamaIsPres
  • Again, use exit() to quit the Python Evaluator

Here we are using python's syntax for setting and using variables. A variable is set in python using variableName = value and the value can be any data type. (Note == compares two variables = sets a variables value.) A variable can be created, set, compared, and changed. Above we can see some basic functionality of variables with the three data types we have seen.

Part 2 - Unix Basics

Creating a Directory

We have been using the command prompt to run the python evaluator, but that is only one program out of thousands that the command prompt can run. Before the mouse and fancy graphic displays were invented, the command prompt was the only way to navigate and manipulate the files on a computer. Below we will walk through a few simple commands to create a new folder for you to put your coding projects in.

  
						% cd                 # changes the current directory to your home directory
						% pwd                # prints out the directory (to verify where you are)
						% mkdir newProject   # creates a directory named 'newProject'
						% ls                 # lists the contents of your current directory
						% cd newProject      # changes your current directory to the one you just made
						% pwd                # prints out your current directory
