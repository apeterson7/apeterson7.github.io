Data Types

Python has a few simple data types that store different types of data, and are manipulated in different ways.

  • Strings: Defined by "". Used to store text. Can be printed, concatonated, compared, and more.
  • Integer: Store numbers, math operators can be applied, can be compared, can be printed
  • Boolean: Can be compared, are produced when two pieces of data of the same type are compared.

All coding languages are documented in their entireity online, if you are interested in reading more about strings, booleans, or integers, feel free. You will not understand everything you read in the documentation-- and that's okay. As a programmer, it is important to learn how to identify relevant information and disregard information that is not relevant to what you are working on.

For Loop

When coding, we often want to repeat the same bit of code multiple times. One way of doing so is with a For Loop. A basic For Loop uses an integer i with initial value 1, and both execute the code within the loop and increment i by 1, until i's value is equal to the specified range. The following Python code will produce the same output.

print(1) print(2) print(3) print(4) print(5)
for i in range 5: print(i)

( Note: whitespace is important in python. the for loop must end with : and the code within the For Loop must be indendented using tab )

If Statement

An if statement is a common type of conditional in programming languages. It takes a condition, for example 5 > 3, and evaluates whether it is True or False. If it is True it will run the code under if, otherwise it will run the code under else.

apples = 3 oranges = 4 if(apples == oranges): print("There are as many apples as oranges") else: print("The numbers of apples and oranges are unequal")

Note: python requires : after if and else statements.
